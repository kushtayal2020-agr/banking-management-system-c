# Banking Management System (C)

A console-based Banking Management System developed using C language.

## Features
- Login system
- Account opening
- Account verification using Account Number & Customer ID
- Deposit and withdrawal
- Account details display

## Technologies Used
- C Programming
- Structures
- Functions
- Conditional statements

## How to Run
Compile using GCC or Turbo C and run the program.
